# 声记播客部署端口说明

## 重要提醒

**应用始终运行在端口 80，不是 8080！**

## 访问地址
- **正确地址**: http://124.220.31.71/
- **错误地址**: http://124.220.31.71:8080/ (不要使用)

## 部署流程

1. **构建项目**
   ```bash
   npm run build
   ```

2. **启动服务**
   ```bash
   npm start
   ```
   
   **注意**: 应用会自动在端口 80 启动，无需指定端口

3. **验证部署**
   ```bash
   # 检查进程
   ps aux | grep next
   
   # 检查端口
   netstat -tlnp | grep :80
   ```

## 常见问题

### 问题：无法访问网站
**原因**: 应用没有在端口 80 运行
**解决**: 
```bash
# 停止现有进程
pkill -f "next start"

# 重新启动
npm start
```

### 问题：端口被占用
**解决**: 
```bash
# 查看占用端口 80 的进程
lsof -i :80

# 如果被其他进程占用，停止该进程或使用 sudo 权限
sudo npm start
```

## 部署脚本

使用 `deploy.sh` 脚本会自动处理所有步骤，包括在端口 80 启动服务。
